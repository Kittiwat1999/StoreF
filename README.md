# E-Commerce Learning Project

This repository contains a simple e-commerce application built as part of a study project.  
The goal is to explore **React (TSX)** for the frontend and **.NET Web API + MSSQL** for the backend, while practicing full-stack development patterns.

## 🎯 Purpose
- Learn how to structure a full-stack application
- Practice React component design, routing, and state management
- Implement RESTful APIs with .NET 8 and Entity Framework Core
- Work with MSSQL for relational data modeling
- Experiment with Docker Compose for local development

## 🛒 Features
- Product listing and detail pages
- Cart management and confirm order flow
- Basic authentication (JWT)
- API integration between frontend and backend
- Database schema for products, users, and orders

## ⚙️ Tech Stack
- **Frontend:** React (TSX), Vite, TailwindCSS
- **Backend:** .NET 8 Web API, EF Core
- **Database:** MSSQL
- **Infra:** Docker Compose, Nginx (reverse proxy)
